# Dotfiles

## About the code

Dotfiles contains bash scripts which allows to personalize git bash for windows.

## Installation

1) Download and extract files into your root directory
2) To display icons - install your favourite nerd font (https://www.nerdfonts.com/)
3) If you would like to customize the look or behaviour of the bash please see comments in each individual file
